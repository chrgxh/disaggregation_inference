# Energy Disaggregation & Forecasting API

FastAPI service for appliance-level energy disaggregation (Gated CNN architecture) and time-series forecasting (GRU), driven by YAML configuration files.

---

## Overview

The API exposes two endpoints:

1. Disaggregation
2. Forecasting

All model paths, thresholds, window sizes, and building metadata are defined in YAML files.

Data is currently loaded from CSV files specified in the configuration.  
The design is extensible and can be adapted to load data from a live database.

---

## Disaggregation Endpoint

Implements a two-stage gated architecture:

- CNN classifier → predicts ON/OFF probability
- CNN regressor → predicts appliance power (ON-only)
- Final output is gated using classifier prediction

Pipeline:

1. Load building data
2. Create sliding windows (+ delta mains)
3. Run classification
4. Run regression
5. Apply gating
6. Return structured JSON response

---

## Forecast Endpoint

GRU-based time-series forecasting.

- Uses last 48 hours as input
- Predicts next 24 hours
- Loads `.h5` model and scaler `.pkl`
- Returns future time series as JSON

---

## Configuration

All settings are defined in YAML files:

- Building CSV paths
- Model paths
- Window sizes
- Thresholds
- Scaling parameters

---

## Run Locally

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

---

## Run with Docker

Build image:

```bash
docker build -t disagg-api .
```

Run container:

```bash
docker run -p 8000:8000 disagg-api
```

API will be available at:

```
http://localhost:8000
```
